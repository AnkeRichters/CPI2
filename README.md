# CPI2
Metastatic Clear-Cell Renal Cell Carcinoma: Clinical Prognostic Index in the Checkpoint Inhibitor era (CPI2)

## Publication:
Pre-print available from: https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6733423

Journal publication pending.


## Using the calculator:

Install R and then run the following in your R console:

```
install.packages("shiny")
library(shiny)
shiny::runGitHub("CPI2", "AnkeRichters")
```
